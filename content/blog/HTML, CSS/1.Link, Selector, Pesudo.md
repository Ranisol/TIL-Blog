# 1. CSS 사용하기
## (1) inline (태그 내) 
- 태그 내 property(color/font-style)을 부여해서 사용 가능
```html
<h1 style="color: red; font-style: italic"> 제목 </h1>
```

## (2) HTML내부(style) 
- 특정 태그에 대해 스타일 일괄 적용하기
- 또는 태그를 선택하는 규칙(Selector)에 따라 일괄 적용하기
```html
<head>
  <style>
    h1{
      color: red;
      font-style: italic;
     }
  </style>
</head>
```
- 보통 head태그 안에 삽입하는 경우가 많음

## (3) HTML외부에서(link)
```html
<head>
  <title></title>
  <link rel="stylesheet" type="text/css" href="Mystyle.css" />
</head>
```


# 2. Selector 여러개 접근하기

## cf) 공백을 이용해서 여러 클래스 지정 가능
```html
<h1 class='red underline'>Hello</h1>
<h1 class='blue'>world</h1>
```


## (1) 모든 요소에 적용하기
```css
* {		
  text-align: center;
  color: blue;
}
```

## (2) 여러개 묶어서 적용하기

```css
h1, h2, p {		여러개 묶어서 가능
  text-align: center;
  color: red;
}
```


# 3. 상속 관계 이용한 Selector

## (1) `후손 요소` 접근하기
div아래의 모든 p를 선택
```html
<style>
  div p { color: red; }
</style>
<p>paragraph 2</p>
<span><p>paragraph 3</p></span>
```

각각의 요소를 id나 class를 사용해서 더 세부적으로 접근 가능.
```css
div#center span.hello{		
   background-color: red;
}
```

## (2) `자식 요소` 접근하기
div 바로 아래의 p들을 선택한다.
```css
div > p {
  color : red
}
```
## (3) `형제 요소` 접근하기
p 바로 뒤에 오는 ul를 선택한다.
```html
<style>
  p + ul { color: red; }
</style>
<p>The first paragraph.</p>
  <ul>
    <li>Coffee</li>
    <li>Tea</li>
    <li>Milk</li>
  </ul>
```
## (4) `형제 요소 전체`접근하기
p 바로 뒤에 오는 ul를 선택한다.
```html
<style>
  p + ul { color: red; }
</style>
<p>The first paragraph.</p>
  <ul>
    <li>Coffee</li>
    <li>Tea</li>
    <li>Milk</li>
  </ul>
```



# 4. 속성과 값을 이용한 Selector

## (1) `태그 + 속성`으로 접근하기
```css
a[href] {
  color: red
}
```

## (2) `태그 + 속성 + 전체 값`
```html
<style>
  a[target="_blank"] { 
    color: red;
  }
</style>
```
## (3) `태그 + 속성 + 독립적인 일부 값`

```html
<style>
h1[title~="first"] {
  color: red; 
}
</style>
<h1 title="heading first">Heading first</h1> /*적용됨*/
<h1 title="heading-first">Heading-first</h1>  /*적용 안됨*/
```

## (4) `태그 + 속성 + 일부 값`
```html
<style>
div[class*="test"] {
   background-color: yellow; 
}
</style>
<div class="first_test">The first div element.</div>
```

## (5) `태그 + 속성 + 시작 값`
```css
a[href^="https://"] {
   color: red; 
}
```

## (6) `태그 + 속성 + 끝부분 값`
a[href$=".css"] {
   color: red; 
}

## (7) `태그 + 속성 + 시작 값 / 값 자체`
```css
<style>
  /* p 요소 중에 lang 어트리뷰트 값이 "en"과 일치하거나 "en-"로 시작하는 요소 */
  p[lang|="en"] { color: red; }
</style>
<p lang="en">Hello!</p>
<p lang="en-us">Hi!</p>
<p lang="en-gb">Ello!</p>
```

# 6. Pesudo-class(가상 클래스)
선택자의 상태에 따른 지정. 상태란?
- 마우스가 올라와 있을 때
- 링크를 방문했을 때, 또는 방문하지 않았을 때 등

## (1) Pesudo-class vs Pesudo-element
유사 클래스는 선택한 요소의 특정 상태일 때 반영. (a태그 사용시, 이미 클릭한 후의 색깔)
유사 요소는 선택한 요소의 일부분에 반영하는 것. (p태그 사용시, 문단의 처음)

## (2) 종류 및 역할
<a href="https://www.w3schools.com/css/css_pseudo_classes.asp">w3school</a>에 다 나와있음. 여기서는 사용할 만한것들 번역

## (3) 링크 셀렉터
### `:link` (방문하지 않은 링크)

### `:visit` (방문한 링크)

## (4) 동적 셀렉터
### `:hover` (마우스가 올라와 있을 때)

### `:active` (클릭된 상태일 때)

### `:focus` (포커스(타이핑 바)가 들어와 있을 때)

## (5) UI요소의 상태에 따른 셀렉터
### `:checked` (셀렉터가 체크 상태일 때)

### `:enabled` (셀렉터가 사용 가능한 상태일 때)

### `:disabled` (셀렉터가 사용 불가능한 상태일 때)
Input박스는 디폴드 속성으로 enabled인 상태인데, 속성에 disabled라고 추가하게 되면, 해당 input을 클릭할 수 없게 됨.
```html
<input type="checkbox" value="motorcycle" disabled> <span>I have a motorcycle</span>
```

## (5) 구조적으로 특정 상태에 따른 셀렉터
### `:first-child` (첫번째 자식인 요소를 선택)

### `:last-child` (마지막 자식인 요소를 선택)

### `:nth-child(n)` (n번째 자식인 요소 선택)

### `:nth-last-child(n)` (뒤에서부터 n번째 자식인 요소 선택)

### `first-of-type` (셀렉터의 부모 요소 중 자식 요소의 첫번째에 등장하는 셀렉터와 동일한 요소)

... first대신에 last, nth, nth-last 붙여주면 다 똑같음.

## (6) 부정 셀렉터
### `:not(셀렉터)` 셀렉터에 해당하지 않는 모든 요소
```css
input:not([type=password]) {
  background: yellow;
}
```
input중 type=password가 아닌 모든 input선택함
