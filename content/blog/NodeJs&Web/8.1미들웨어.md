# 미들웨어 사용하기

## 미들웨어 불러와서 사용

```js
const cors = require('cors')
app.use(cors())
```

## 내가 정의한 미들웨어 사용

```js
const middleWare = (req, res, next) => {
    console.log(`시작을 알리는 미들웨어:${req.protocol}://${req.get('host')}${req.originalUrl}에서 실행`)
    next()
}
app use(middleWare)
```
위의 middleWare를 다른 파일에서 정의하고, require('middleWare')를 통해 불러와서 사용할 수 있음.
